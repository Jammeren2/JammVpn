# JammVPN

**JammVPN** — десктопный VPN-клиент для Windows с продвинутым **раздельным туннелированием** (split tunneling): через VPN направляется трафик только выбранных приложений, при этом локальная сеть и программы, не добавленные в список, **полностью не затрагиваются**.

> ⚠️ **Статус:** ранняя стадия — проектирование / pre-alpha. Архитектура и спецификация в активной проработке.

## Ключевые возможности

- 🔀 **Точное раздельное туннелирование (per-app split)** — два режима:
  - **инклюзивный** — в VPN идёт трафик только выбранных приложений;
  - **эксклюзивный** — в VPN идёт весь трафик, кроме выбранных приложений.
  - Локальная сеть (LAN, роутер, принтеры, сетевые папки) и невыбранные приложения **никогда** не заворачиваются в тоннель.
- 🧩 **Множество протоколов** (нативная реализация на Rust):
  - VLESS (REALITY, XTLS-Vision; транспорты TCP / WS / gRPC / XHTTP)
  - Shadowsocks (AEAD, SS-2022)
  - WireGuard и **AmneziaWG** (обфусцированный WG)
  - SOCKS5, HTTP(S)-прокси
  - Trojan, Hysteria2, TUIC v5
  - Опционально — внешнее ядро **sing-box** как сменный бэкенд.
- 🛡 **Защита:** kill-switch, защита от DNS / IPv6 / WebRTC-утечек, обфускация и обход DPI (отключаемо).
- 🌐 **Умная маршрутизация:** правила по домену / IP / процессу / гео (geosite, geoip), готовые пресеты, редактор правил.
- 🧭 **DNS:** FakeIP + DoH / DoT внутри тоннеля.
- 📥 **Импорт конфигов:** ссылки (`vless://`, `ss://`, `wg`…), base64-подписки с автообновлением, Clash / Clash.Meta YAML, Xray / sing-box JSON, конфиг AmneziaWG из файла, QR-код, ручной ввод.
- 🔒 **Приватность:** нулевая телеметрия; секреты шифруются через Windows DPAPI.

## Технологии

- **Ядро / логика:** Rust — переносимый крейт `core` (протоколы, маршрутизация, движок split).
- **Платформенный слой (Windows):** WFP (user-mode connect-redirect), TUN-адаптер (wintun).
- **UI:** [Tauri](https://tauri.app/) (webview) + веб-фронтенд.
- **Архитектура:** чистое разделение `core ↔ platform` — задел под будущий перенос на Android.

## Поддерживаемые ОС

- **Windows 10 (1809+) и Windows 11**, x64 (ARM64 — в планах).
- Android — рассматривается в будущем (ядро держим переносимым).
