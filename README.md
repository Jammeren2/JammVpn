# JammVPN

**JammVPN** — десктопный VPN-клиент для Windows с продвинутым **раздельным туннелированием** (split tunneling): через VPN направляется трафик только выбранных приложений, при этом локальная сеть и программы, не добавленные в список, **полностью не затрагиваются**.

## Установка

1. Скачайте `jammvpn-app.exe` из [последнего релиза](https://github.com/Jammeren2/JammVpn/releases/latest).
2. Запустите его — **всё в одном файле**, установка не требуется. Фронтенд и драйвер
   раздельного туннелирования вшиты в exe.
3. Для раздельного туннелирования нужны права администратора — приложение само
   предложит перезапуститься от админа и при первом включении split автоматически
   установит свой NDIS-драйвер (WinpkFilter, подпись Microsoft, без перезагрузки).

> Требуется **WebView2 Runtime** (предустановлен в актуальных Windows 10/11; иначе
> ставится автоматически из Microsoft Update).

### Обновления

Приложение само проверяет последний релиз на GitHub при запуске и, если доступна
новая версия, показывает баннер «Обновить». По нажатию новый `.exe` скачивается,
подменяет текущий и приложение перезапускается — вручную ничего качать не нужно.

## Ключевые возможности

- 🔀 **Точное раздельное туннелирование (per-app split)** через драйвер **WinpkFilter**
  (захват на уровне NDIS, как у WireSock) — и TCP, и UDP выбранных приложений идут
  через тоннель, остальной трафик и LAN не затрагиваются. Два режима:
  - **инклюзивный** — в VPN идёт трафик только выбранных приложений;
  - **эксклюзивный** — в VPN идёт весь трафик, кроме выбранных приложений.
- 🧩 **Множество протоколов** (нативная реализация на Rust):
  - VLESS (REALITY, XTLS-Vision; транспорты TCP / WS / gRPC / XHTTP)
  - Shadowsocks (AEAD, SS-2022)
  - WireGuard и **AmneziaWG** (обфусцированный WG)
  - SOCKS5, HTTP(S)-прокси
  - Trojan, TUIC v5
  - Локальный **WireGuard-сервер** (раздать тоннель на другие устройства, экспорт `.conf` и QR).
- 🛡 **Защита:** kill-switch, защита от DNS / IPv6-утечек.
- 🌐 **Умная маршрутизация:** правила по домену / IP / процессу / гео (geosite, geoip,
  авто-загрузка баз), редактор правил.
- 🧭 **DNS:** FakeIP внутри тоннеля.
- 📥 **Импорт конфигов:** ссылки (`vless://`, `ss://`, `trojan://`, WG…), base64- и
  JSON-подписки с автообновлением, Clash / Clash.Meta YAML, Xray / sing-box JSON,
  конфиг AmneziaWG из файла, ручной ввод.
- 🔒 **Приватность:** нулевая телеметрия; секреты шифруются через Windows DPAPI.

## Технологии

- **Ядро / логика:** Rust — переносимый крейт `core` (протоколы, маршрутизация, движок split).
- **Сеть:** асинхронное ядро на `tokio` — outbound-диалеры, локальный прокси,
  userspace-netstack (tun2socks через smoltcp) для split и локального WG-сервера.
- **Платформенный слой (Windows):** WinpkFilter (NDIS LWF) для per-app split, WFP, DPAPI.
- **UI:** [Tauri 2](https://tauri.app/) (webview) + vanilla веб-фронтенд.
- **Архитектура:** чистое разделение `core ↔ platform` — задел под будущий перенос на другие ОС.

## Поддерживаемые ОС

- **Windows 10 (1809+) и Windows 11**, x64 (ARM64 — в планах).

## Структура репозитория

```
crates/
  core/               # переносимое ядро: модель, парсеры импорта, split/routing-движки
  net/                # асинхронное сетевое ядро (tokio): outbound, netstack, локальный WG-сервер, подписки
  platform-windows/   # Windows-слой: WinpkFilter split + вшитый драйвер, WFP, DPAPI
  cli/                # контроллер (вся логика поверх ядра; используется GUI)
app/
  src-tauri/          # Tauri-приложение (команды, трей, окно)
  ui/                 # фронтенд (HTML/CSS/JS)
wfp-driver/           # legacy kernel-mode WFP callout-драйвер (скелет)
```

## Сборка

Требуется Rust (stable). Сборка продакшн-бинарника (самодостаточный exe):

```sh
cargo build --release -p jammvpn-app
# результат: target/release/jammvpn-app.exe
```

Тесты и линт:

```sh
cargo test  --workspace
cargo clippy --all-targets -- -D warnings
cargo fmt --all -- --check
```
